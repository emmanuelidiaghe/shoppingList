Shopping App

Welcome to Emancipator Embedded Grocery Store...

This is a shopping web app that allows you create a list of items to be purchased with pre-existing items.
The following can be done with the app:

  1. Type in the item to the purchased and click the Enter button or press the Enter key
  2. Click on a particular item to mark it purchased
  3. Click the Delete button to remove a corresponding item on the list
  4. Choose two colors from the color inputs to set the background color gradient
  5. Happy Shopping!

Please select items to be purchased:

  • Pancakes
  • NoteBooks
  • Pyjamas
